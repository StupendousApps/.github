# StupendousApps

We build stupidly powerful software.

---

## Repos

### Organization

* [**.github**](https://github.com/StupendousApps/.github) — org profile and shared templates
* [**StupendousAgents**](https://github.com/StupendousApps/StupendousAgents) — shared agent guidance and style catalogue

### Infrastructure
* [**stupendous_monitor**](https://github.com/StupendousApps/stupendous_monitor) — shared ops stack on the Kimsufi (Beszel dashboard + Umami analytics)
* [**stupendous_static**](https://github.com/StupendousApps/stupendous_static) — single Caddy container serving every Stupendous static site (`stupendousapps.com`, `fceruti.com`, …)

### Projects
* [**Dotty**](https://github.com/StupendousApps/Dotty) — native macOS app

### Web
* [**dotty_web**](https://github.com/StupendousApps/dotty_web) — Phoenix/Elixir app for dotty.app (marketing, community, licenses, admin)
* [**macatlas**](https://github.com/StupendousApps/macatlas) — Rails app for macatlas.com, a curated macOS app catalogue
* [**lets_talk_art**](https://github.com/StupendousApps/lets_talk_art) — Phoenix/Elixir app for letstalk.art

### Libraries
* [**DottyModels**](https://github.com/StupendousApps/DottyModels) — private/local data models
* [**DottyRichText**](https://github.com/StupendousApps/DottyRichText) — shared rich-text toolkit
* [**DottyCommunityClient**](https://github.com/StupendousApps/DottyCommunityClient) — client SDK for public APIs
